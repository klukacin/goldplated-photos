# Card Block Images

Place card images for each section here:
- **Files needed:**
  - `weddings.jpg`
  - `people.jpg`
  - `events.jpg`
  - `street.jpg`
  - `markusevec.jpg`
  - `friends.jpg`

- **Source:** Extract from https://k.lukac.in/digital-home
- **Size:** Rectangular/landscape format, at least 800px wide
- **Quality:** Good resolution for display in content cards

## How to get images:
1. Visit https://k.lukac.in/digital-home
2. Find the images for each section (Weddings, People, Events, etc.)
3. Right-click and save each image
4. Rename them to match the filenames above
5. Place in this folder

These images will be displayed alongside the text content for each section.
