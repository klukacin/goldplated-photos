# Hero Slider Images

Place 3 wide hero images here:
- **Files:** `image1.jpg`, `image2.jpg`, `image3.jpg`
- **Source:** Extract from https://k.lukac.in/digital-home
- **Size:** Wide format (16:9 or similar), at least 1920px wide
- **Quality:** High resolution for slider display

## How to get images:
1. Visit https://k.lukac.in/digital-home
2. Right-click on the hero/banner images
3. Save as `image1.jpg`, `image2.jpg`, `image3.jpg`
4. Place in this folder

These will be used in the auto-rotating hero slider.
