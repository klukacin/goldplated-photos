# Shutter Sound Effect

Place a camera shutter sound file here:
- **File:** `shutter.mp3`
- **Source:** Free sound effect from freesound.org or similar
- **Duration:** Short (0.2-0.5 seconds)
- **Format:** MP3

## Where to find:
- https://freesound.org/ (search for "camera shutter")
- https://mixkit.co/free-sound-effects/camera/
- Any royalty-free sound effect library

Make sure the license allows free use!
