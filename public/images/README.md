# Landing Page Background Image

Place the landing page background image here:
- **File:** `landing-bg.jpg`
- **Source:** Download from https://k.lukac.in/
- **Size:** High resolution (at least 1920x1080)

## How to get the image:
1. Visit https://k.lukac.in/
2. Open browser DevTools (F12)
3. Go to Network tab
4. Look for the background image being loaded
5. Download it and save as `landing-bg.jpg` in this folder
